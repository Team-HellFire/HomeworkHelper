# 프로젝트 제목 
- ### HomeworkHelper
- 대학생들의 과제 해결에 필요한 기능을 모아 제공하는 웹 서비스

# 프로젝트의 배경
- 최근 코로나 펜데믹으로 인해 대면 수업과 비대면 수업을 번갈아 하며 늘어난 과제량으로 타임푸어 현상을 겪는 대학생들을 위해 과제에 필요한 기능을 한 곳에 모아 사용하게 함으로 과제에 사용되는 불필요한 시간 해소

# 사용된 API 및 라이브러리
 - Papago 번역 API 
 - RISS 논문 검색 API
 - hanspell 라이브러리
 - ko-spellchecker 라이브러리
